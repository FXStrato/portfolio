# Preact/Parcel
## Getting Started
1. Pre-reqs: [Yarn](https://yarnpkg.com) | [NodeJS](https://nodejs.org)
2. Clone repo
3. Run ```yarn``` then ```yarn start``` from project root

## Building
1. Run ```yarn build```
2. Output files will be in dist folder

## Troubleshooting
1. Run ```yarn clean``` to clear local cache and remove old builds
